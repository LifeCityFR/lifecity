DefCon Niveau de Sécurité

Code VERT : Aucun problème, aucun danger.
Les patrouilles de police peuvent patrouiller normalement.
Code JAUNE : Quelques petites interventions basiques.
Les patrouilles de police patrouillent normalement.
Code ORANGE : Interventions à risques élevés.
Les patrouilles peuvent contrôler les véhicules sans raisons précises. Le SWAT et le GANG UNIT peuvent patrouiller.
Code ROUGE : Grosse quantité d'interventions à haut risques.
Les patrouilles peuvent fouiller et contrôler tous les véhicules et civils correspondant à un suspect sans motifs.
Code NOIR : Danger extrême pour les citoyens/policiers.
Exemple : Attaque terroriste, alerte à la bombe, etc.
